## Installing and run it 

```bash
git clone https://github.com/iamaloneforever/mohsenian-tech.git

cd mohsenian-tech

# Install Dependencies 

#pnpm
pnpm install
#npm 
npm install

# Run

#pnpm
pnpm dev

#npm 
npm run dev

```
